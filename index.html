Introducción a CSS

Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de documentos HTML. Permite controlar el diseño visual, como colores, fuentes, y el posicionamiento de elementos en una página web.

Funcionamiento Básico de CSS

CSS funciona aplicando estilos a elementos HTML según las reglas definidas por el autor. Estas reglas se pueden aplicar de manera directa en el archivo HTML, dentro de etiquetas <style>, o en archivos CSS externos.

Incluir CSS en un archivo HTML

  • Interno: Dentro de la etiqueta <style> en el <head>.
  • Externo: Mediante la etiqueta <link> en el <head> para enlazar un archivo .css.
  • En línea: Directamente en la etiqueta HTML usando el atributo style.

Definición de una regla CSS

Una regla CSS define cómo se debe mostrar un elemento específico en la página. Incluye un selector que apunta al elemento y un bloque de declaración que define los estilos.

Elementos de una regla CSS

  • Regla: Conjunto de una o más declaraciones que aplican estilos.
  • Selector: Parte que define a qué elemento(s) se aplicará la regla.
  • Bloque de declaración: Contiene las propiedades y valores que definen los estilos.
  • Propiedad: Característica de diseño que se quiere modificar (ej. color, margen).
  • Valor: Especifica cómo se debe aplicar la propiedad (ej. rojo, 10px).

Medios CSS

Son reglas que aplican estilos según las características del dispositivo, como el ancho de pantalla, usando consultas de medios (media queries).

Comentarios CSS

Son notas que no afectan el código y se usan para documentar. Se escriben entre /* comentario */.

Sintaxis de la definición de las propiedades CSS

Las propiedades se definen dentro de un bloque de declaración en el formato propiedad: valor; donde "propiedad" es lo que se quiere modificar y "valor" es el cambio aplicado.
